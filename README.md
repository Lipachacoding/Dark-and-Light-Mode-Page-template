# Dark-and-Light-Mode-Page-template for a Medical Hospital Dashboard
#This project is a hospital report system user interface that demonstrate DOM 
manipulation techniques using JavaScript.

#Dynamic theme switching.
- Hospital dashboard layout.
- Responsive grid system.
- DOM manipulation using:
    -getElementById()
    -addEventListener()
    -classList.toggle()
    -textContent updates.
  #How it works
  -By clicking the toggle button it:
    -switches the css class on the body element.
    -The entire color scheme changes.
    -updates the button text dynamically.
